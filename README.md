# 🌐 Nithiyanantham K — Personal Portfolio

![Portfolio Banner](imgs/myimg1.jpeg)

> A sleek, dark-themed personal portfolio website built with pure **HTML**, **CSS**, and **JavaScript** — showcasing my skills, projects, and experience as a Web Developer & Data Analyst.

---

## ✨ Features

- 🎨 **Dark luxury UI** with warm gold-tan accent color scheme
- 💫 **Smooth fade-in animations** using Intersection Observer API
- 🔄 **Rotating hero role text** (Web Developer → Data Analyst → Full Stack Developer → AI & ML Enthusiast)
- 📱 **Fully responsive** — mobile, tablet, and desktop
- 🍔 **Mobile hamburger menu** with smooth toggle
- ⚡ **Zero dependencies** — pure HTML, CSS, JS (no frameworks)
- 🌀 **Animated profile ring** with counter-rotating image effect
- 🖱️ **Smooth scroll** navigation

## ☁️ Deployment

### 🐙 Deploy to GitHub Pages

1. **Create a GitHub repository**
   - Go to [github.com](https://github.com) → **New Repository**
   - Name it: `portfolio` (or `YOUR_USERNAME.github.io` for a root site)
   - Set visibility to **Public**

2. **Push your code**
   ```bash
   git init
   git add .
   git commit -m "🚀 Initial portfolio commit"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/portfolio.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repo → **Settings** → **Pages**
   - Under *Source*, select **Deploy from a branch**
   - Branch: `main` | Folder: `/ (root)`
   - Click **Save**

4. **Access your site**
   ```
   https://YOUR_USERNAME.github.io/portfolio
   ```
   > ⏱️ Takes ~2 minutes to go live.

---

### ▲ Deploy to Vercel (Recommended — Faster & Custom Domain)

1. **Push to GitHub** (follow steps above first)

2. **Connect to Vercel**
   - Go to [vercel.com](https://vercel.com) → **Sign up with GitHub**
   - Click **"Add New Project"**
   - Import your `portfolio` repository

3. **Configure project**
   - Framework Preset: **Other** (static site)
   - Root Directory: `./`
   - No build command needed

4. **Deploy**
   - Click **Deploy** — Vercel builds and deploys automatically
   - You'll get a live URL like:
     ```
     https://portfolio-YOUR_USERNAME.vercel.app
     ```

5. **Auto-deploy on push**
   - Every `git push` to `main` auto-redeploys your site 🎉
